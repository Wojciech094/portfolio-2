# Portfolio 2 — Wojtek Leśniak

A multipage frontend development portfolio created for the **Portfolio 2** assignment at Noroff. The website presents three selected course projects, the improvements made to each project, and links to the deployed applications and their documentation.

## Live Demo

[View Portfolio Website](https://portfolio2woj.netlify.app/)

## Featured Projects

### Social Media Client — CSS Frameworks

A responsive social media interface built with TypeScript, Vite and Tailwind CSS. The Portfolio 2 improvement focused on improving mobile responsiveness, refining the post details presentation and correcting author display issues.

* Live Site: `(https://bucolic-mousse-4e51e9.netlify.app/login)`
* Repository: `(https://github.com/Wojciech094/Social-Media-App)`

### React Online Shop — JavaScript Frameworks

An e-commerce application built with React and TypeScript. The Portfolio 2 improvement focused on creating a safer checkout flow and improving the layout of short pages.

* Live Site: `(https://taupe-sopapillas-2582ff.netlify.app/)`
* Repository: `(https://github.com/Wojciech094/React-online-shop)`

### BidHouse Auction Marketplace — Semester Project 2

A responsive auction platform built with HTML, Tailwind CSS and vanilla JavaScript. The Portfolio 2 improvement focused on adding reliable fallback images for missing or unavailable listing media.

* Live Site: [BidHouse Live Demo](https://bidhouseweb.netlify.app/)
* Repository: [BidHouse GitHub Repository](https://github.com/Wojciech094/BidHouse)

## Getting Started

### Prerequisites

Make sure you have installed:

* Node.js version 18 or newer
* npm

### Installation

1. Clone this repository:

   ```bash
   git clone https://github.com/Wojciech094/portfolio-2.git
   ```

2. Navigate into the project directory:

   ```bash
   cd portfolio-2
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

4. Start the development server:

   ```bash
   npm run dev
   ```

5. Open the local URL shown in the terminal.

## Build for Production

To create a production build:

```bash
npm run build
```

To preview the production build locally:

```bash
npm run preview
```
